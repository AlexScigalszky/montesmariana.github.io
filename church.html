Welcome to Church. Here you will find a table with the following clouds:

  • Only of the token church.
  • From Semcor, with Semcor as source of the first order weighting values.
  • With either COHA or Semcor as source of the second order values. In all cases the same context words were used, only the ppmi values differ. These are all the 5795 content words with frequency over 10 in both corpora.
  • Without any other kind of subsetting: these are all the tokens.

Each column indicates a different threshold in the first order weighting, while each row indicates a different threshold in the second order weighting.
